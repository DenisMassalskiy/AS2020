Для того чтобы создать базу данных необходимо:
1.Скачать скрипт с базой.
2.Запустить MS SQL Management Studio
3.Комбинацией Ctrl+N создать новый запрос.
4.Вставить скрипт в запрос и выполнить его.

Для запуска приложения нужно:
1.Скачать папку с программой с Яндекс Диска https://yadi.sk/d/-LaDd2fO38RM-g
2.Зайти в папку Dron Taxi\bin\Debug
3.Запустить файл Dron Taxi.exe
